# News Sentiment & Quantitative Financial Analysis

A data science project focused on exploratory analysis of financial news data and quantitative stock market analysis using technical indicators.

The project combines Natural Language Processing (NLP), exploratory data analysis (EDA), and financial time-series analysis to understand patterns in financial news and stock market behavior.

---

## Project Objectives

### Task 1 — Exploratory Data Analysis (EDA)
Analyze financial news data to identify:

- Publication trends over time
- Most active publishers
- Common keywords and recurring topics
- Temporal patterns in news activity

### Task 2 — Quantitative Financial Analysis
Analyze historical stock price data using technical indicators such as:

- Simple Moving Average (SMA)
- Exponential Moving Average (EMA)
- Relative Strength Index (RSI)
- Moving Average Convergence Divergence (MACD)
- Rolling Volatility
